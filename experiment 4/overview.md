## Aim
To practice SQL queries for retrieving, updating, and manipulating employee data using various conditions, operators, and functions.

## Theory
Structured Query Language (SQL) is used to interact with databases. It allows users to perform operations such as retrieving data (SELECT), updating data (UPDATE), filtering records (WHERE), and performing calculations.

In this experiment, we use:
- Logical operators (AND, OR, NOT)
- Pattern matching (LIKE)
- Arithmetic operations
- Sorting (ORDER BY)
- Conditional filtering
