# Experiment: Retrieving Data from Employee Table

## Aim
To perform various SQL queries on the Employee table to retrieve specific data using conditions, filtering, and operators.

## Theory
SQL (Structured Query Language) is used to interact with databases. The SELECT statement is used to fetch data from a table. Various clauses like WHERE, DISTINCT, BETWEEN, and logical operators (AND, OR, NOT) help in filtering and retrieving meaningful information from the database.
