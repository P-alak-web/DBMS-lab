# Experiment: Advanced Data Retrieval from Employee Table

## Aim
To perform advanced SQL queries using sorting, pattern matching, logical conditions, and arithmetic operations on the Employee table.

## Theory
SQL provides powerful features to retrieve and manipulate data. Clauses such as ORDER BY are used for sorting, LIKE is used for pattern matching, and logical operators (AND, OR) help in applying multiple conditions. Arithmetic operations can also be performed on columns to derive new values like total salary and annual salary.
