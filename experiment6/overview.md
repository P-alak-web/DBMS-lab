# SQL Lab Manual (Date Functions & Queries)

## Aim
To understand and implement SQL queries using date functions, formatting functions, and conditional logic on employee data.

## Theory
SQL provides various functions to manipulate and format dates such as:
- SYSDATE: Returns current system date
- TO_CHAR(): Converts date to string format
- TO_DATE(): Converts string to date
- MONTHS_BETWEEN(): Finds difference between two dates in months
- NEXT_DAY(): Finds next specific weekday
- DECODE(): Conditional function similar to IF-ELSE

These functions help in extracting meaningful information from date fields and performing calculations.
